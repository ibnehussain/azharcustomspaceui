# Testspace

This is a test Copilot Space for exploring the custom MCP UI.

## Purpose

- Test GitHub Copilot Spaces via MCP server
- Validate the custom chat UI
- Experiment with context-aware AI responses

## Tech Stack

- Python / FastAPI
- MCP (Model Context Protocol)
- GitHub Copilot Spaces API
- GitHub Models (gpt-4o)

## Getting Started

1. Clone the repo
2. Set your `GITHUB_TOKEN` in `.env`
3. Run `python src/api_bridge.py`
4. Open `http://localhost:3002`
